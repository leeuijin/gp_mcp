==============================================================================
사전 준비 (Anthropic Console LLM API key) 발급
==============================================================================

MCP 기능을 이용하기 위해서는 Claude 에 가입하여 API 키를 발급받고 유료 가입이 필요합니다.
일반적인 chat 서비스는 무료이지만 LLM API 연동을 위해서는 5.5$(부가세 포함) 지불이 필요합니다.

발급 방법

1. Anthropic Console 접속 및 가입
가장 먼저 개발자용 콘솔 사이트에 접속해야 합니다. (일반 claude.ai 채팅 사이트와 다릅니다.)
Anthropic Console (console.anthropic.com) 에 접속합니다.
이메일 주소를 입력하거나 Google 계정으로 가입(Sign up) 합니다.
휴대폰 번호 인증을 진행합니다. (필수 절차이며, 한국 번호 +82를 선택하고 010-xxxx-xxxx 형식으로 입력하면 문자가 옵니다.)

2. 결제 수단 등록 및 크레딧 충전 (중요!)
최근 정책 변경으로 인해, API 키를 생성하더라도 최소 $5 이상의 크레딧을 미리 충전(Pre-paid) 하지 않으면 키가 작동하지 않는 경우가 많습니다.
대시보드 상단 메뉴 중 [Settings] 또는 [Billing] 탭을 클릭합니다.
Payment Methods에서 해외 결제가 가능한 신용/체크카드를 등록합니다.
Add Credits(크레딧 충전) 버튼을 눌러 최소 금액(보통 $5)을 결제합니다.
참고: API는 쓴 만큼 차감되는 방식이지만, 선불로 잔고를 채워둬야 작동합니다.

3. API Key 생성 (Get API Keys)
결제 준비가 끝났다면 이제 키를 만듭니다.
상단 메뉴 중 **[API Keys]**를 클릭하거나 대시보드 중앙의 [Get API Keys] 버튼을 누릅니다.
우측 상단의 [Create Key] 버튼을 클릭합니다.
키의 이름(Name)을 정하라고 나오는데, 아무거나 입력해도 됩니다. (예: Anthropic_API_Key )
